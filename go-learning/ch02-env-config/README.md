## 环境配置

golang 程序安装链接：https://golang.google.cn/dl/

常用开发工具：vim、emacs、sublime text、atom、vscode(推荐1)、GoLand(推荐2)

环境配置：

* `GOROOT`：go 的安装目录，设置这个环境变量自定义 go 路径
* `GOPATH`：go 的工作目录（项目目录），编译或运行时从这个环境变量中去查找包、依赖
